This project's code convention abide by the following resources:

## Global convention
- Indentation: should be 4 spaces.
- Commit message prefix should include the issue ID. 
    + E.g. #2, prefix, additional prefix: commit message

## Java
- [Google Java Code Convention](https://google.github.io/styleguide/javaguide.html)

## JavaScript/TypeScript
- [Google TypeScript Code Convention](https://google.github.io/styleguide/tsguide.html)

## Commit
- [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/)

We expect you to follow this format to ensure that the codespace is easy to read and and friendly to another contributors. Any contributions that do not follow this format might be rejected.
